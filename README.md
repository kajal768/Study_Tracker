# Study Tracker

Study Tracker is a modern productivity and habit management web application designed to help students and developers organize study routines, track progress, and maintain consistency.

The application combines task management, monthly goal tracking, focus sessions, progress analytics, and scheduled reminders within a clean SaaS-inspired interface.

---

## Features

### Task Management

* Create and manage daily study tasks
* Mark tasks as completed
* Organize tasks by category

### Monthly Goal Tracking

* Set monthly learning targets
* Monitor completion progress visually
* Track consistency over time

### Smart Reminder System

* Schedule study reminders
* Browser notification support
* Alarm popup with sound alerts

### Pomodoro Focus Timer

* Built-in focus session timer
* Start, pause, and reset functionality
* Productivity-oriented workflow

### Progress Analytics

* Interactive monthly progress graph
* Visual productivity insights using Chart.js

### Notes & Planning

* Save daily notes and study reflections
* Maintain next-day planning records

### User Experience

* Responsive modern UI
* Dark / Light mode support
* Glassmorphism-inspired design system

## Getting Started

### Clone Repository

```bash

```

### Open Project

```bash
cd Study_Tracker
```

### Run Application

Open `index.html` in any modern web browser.

---

## Storage

The current version uses browser `localStorage` for storing:

* Tasks
* Goals
* Notes
* Reminder schedules
* Progress data

No backend integration is required for the current release.
